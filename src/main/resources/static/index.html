Hello there

Welcome to the dropwizard + angular + webjars seed project, this project is a mix of:

  • The Dropwizard tutorial
  • The AngularJS tutorial
  • Example of setting up:
    • Swagger at /swagger/
    • Dropwizard admin at /admin/
    • static files are served from the classpaht out of /static/
    • Webjars are served from the classpath out of /webjars/
    • REST resources everywhere else.
  • Example of how to inject singletons and per-request objects using hk2
  • Example of how to use webjars rather than rely on The Internet for your client side library needs, which is strangely hard to find example code for.
